# NONMEM Migration Use Case

Complete workflow for migrating NONMEM models to OpenPKPD.

## Workflow Steps

1. Import NONMEM control file
2. Import CDISC data
3. Validate simulation matches NONMEM
4. Re-estimate with FOCE-I
5. Generate VPC
6. Compare results

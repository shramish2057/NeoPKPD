# Basic NCA Analysis

Single-subject non-compartmental analysis computing standard PK metrics.

## Computed Metrics

| Metric | Description |
|--------|-------------|
| Cmax | Maximum concentration |
| Tmax | Time of maximum |
| AUC_0_t | AUC to last observation |
| AUC_0_inf | AUC extrapolated to infinity |
| t_half | Terminal half-life |
| CL/F | Apparent clearance |
| Vz/F | Apparent volume |

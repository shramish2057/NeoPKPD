# Model Comparison

Comparing models using AIC/BIC and likelihood ratio tests.

## Criteria

| Criterion | Formula | Use |
|-----------|---------|-----|
| AIC | -2LL + 2k | General comparison |
| BIC | -2LL + k*ln(n) | Large sample sizes |
| LRT | ΔOFV vs χ² | Nested models |

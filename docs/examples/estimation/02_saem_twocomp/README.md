# SAEM Two-Compartment Estimation

Parameter estimation using SAEM algorithm for a two-compartment model.

## Method

SAEM (Stochastic Approximation Expectation Maximization) is useful for:
- Complex models with many parameters
- Multi-modal posterior distributions
- Models where FOCE may struggle

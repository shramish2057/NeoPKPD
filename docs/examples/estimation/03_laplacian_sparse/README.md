# Laplacian Estimation for Sparse Data

Parameter estimation using Laplace approximation for sparse pediatric data.

## Use Case

Laplacian estimation is ideal for:
- Sparse sampling designs
- Pediatric populations with limited samples
- Binary or categorical outcomes
